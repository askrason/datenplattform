# ADR-006: Umbrella Chart with YAML Anchors for Security

## Status
Accepted

## Context
The platform has 10+ components, each with Pod/Container security contexts.
Consistency is critical: a missed `runAsNonRoot: true` is a security gap.

## Decision
One Umbrella Chart with shared YAML anchors (BundesMessenger pattern).
All components reuse `&defaultSecurityContext` and `&defaultPodSecurityContext`.
Exceptions are explicit and commented.

## Consequences
- ✅ Consistent security baseline
- ✅ No duplicated security configs
- ✅ Easier to audit compliance
- ✅ Single point of security policy
- ⚠️ Changes to anchors affect all components
- ⚠️ YAML anchors less familiar to some developers
